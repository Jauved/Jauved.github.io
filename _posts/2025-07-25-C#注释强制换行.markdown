# C#注释强制换行

## 00 处理方法

如果你需要在注释中强制换行, 换段, 请参考以下代码块.

其中`<br/>`是指`break`, 强制断行.

用`<para>`和`</para>`

```c#
/// <summary>
/// <para>
/// 第一段第一行内容<br/>
/// 第一段其他内容
/// 第一段其他内容
/// </para>
/// <para>
/// 第二段第一行内容<br/>
/// 第二段其他内容
/// 第二段其他内容
/// </para>
/// </summary>
```
